# 🧠 Explainable Deep Ensemble Framework for Skin Cancer Diagnosis and Risk-Aware Clinical Decision Support

This project focuses on building a deep learning-based ensemble framework for **skin lesion classification** to assist in early skin cancer detection and risk-aware clinical decision-making. The model leverages medical image analysis to classify skin lesions into different categories using deep learning techniques.

---

## 📊 Dataset

We used the **ISIC 2019 Skin Lesion Dataset** for training and evaluation.

📁 Dataset Link:  
https://www.kaggle.com/datasets/salviohexia/isic-2019-skinlesion-images-for-classification

---

## 🎯 Objective

- To classify skin lesions using deep learning models
- To improve diagnostic accuracy using an ensemble approach
- To provide explainable AI outputs for clinical trust and transparency
- To support early detection of skin cancer
